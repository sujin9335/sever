1. 자바 소스 파일
	- 사람이 이해할 수 있는 언어로 구성된 파일
	- Hello.java
	
2. 컴퓨터
	- 1, 0 만 이해할 수 있는 구조
	
3. 자바 소스 파일 > (변환) > 1,0으로 된 파일

4. 번역 프로그램 > Compile(컴파일)
	- javac.exe > Compiler(컴파일러)
	- java compiler
	- javac.exe 파일명.java
	- 산출물 > 파일명.class > Hello.class > 실행 파일
	- 1차 컴파일(1차 번역) > 아직까지 1,0으로 번역 안함 > 중간 언어(IL)로 번역

5. 한번 더 번역 
	- 2차 컴파일(번역) + 번역 결과 실행!!
	- 2차 컴파일(X) > 실행(O)
	- java.exe 파일명
	- java.exe 클래스명
	- Hello.class(IL) > (번역) > 1,0(기계어(Machine Code), 바이너리 코드)
	- 산출물 없음 > 실시간 번역
	- JIT(Just in Time) Compile
